# YW Product Planner

Use this skill before implementation when a request changes product behavior, user flow, data meaning, or milestone scope.

## Output

- Goal and non-goals.
- User-facing behavior.
- Data impact.
- UI/device impact.
- Acceptance criteria.
- Open questions only when they materially change the implementation.
