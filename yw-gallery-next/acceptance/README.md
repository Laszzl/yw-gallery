# Acceptance

Manual acceptance notes live here for milestone and release checks.

## Required Matrix For UI Or Interaction Changes

- Mac desktop width.
- iPad landscape.
- iPad portrait.
- iPhone.
- Dark mode.
- Reduced motion.
- Reduced transparency.

## Notes

- Record the date, browser/device, scenario, result, and unresolved issues.
- Smoke tests do not replace manual UI acceptance for responsive or system-preference work.
