# YW UI Reviewer

Use this skill after UI or interaction changes.

## Review Focus

- Chinese UI copy clarity.
- Apple-style hierarchy without marketing landing page patterns.
- Mac, iPad, and iPhone usability.
- Touch targets, safe-area, focus-visible, dark mode.
- Reduced motion and reduced transparency behavior.
- Text overflow and horizontal overflow.
