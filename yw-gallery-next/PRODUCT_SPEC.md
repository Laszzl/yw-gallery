# YW Gallery Next Product Spec

## 目标

YW Gallery Next 是一个本地优先的中文单页网页，用来记录、浏览和管理 YW 收藏数据。新版允许重新设计 UI、交互和功能优先级，但需要保持用户数据可迁移、核心流程清晰、三端体验可靠。

## 核心用户流程

- 选择人物并进入个人空间。
- 按大品类、小品类浏览 YW。
- 区分有图 YW 和无图 YW，并支持状态、数量、日期等信息。
- 添加、编辑、删除 YW。
- 管理人物、分类、图片、画廊和本地数据。
- 导入旧版导出 JSON，并迁移为新版数据。
- 导出新版 JSON 作为备份。

## 新版取舍

- 不要求复刻旧版页面结构。
- 不要求沿用旧版所有字段名和内部模块。
- 保留旧数据迁移导入，不保证旧版和新版内部模型完全一致。
- 优先让主流程稳定，再逐步加入图片、裁剪、排序和高级管理能力。

## 非目标

- 不做账号体系。
- 不做云同步。
- 不做服务端 API。
- 不引入框架、打包器或需要编译的依赖。
- 不在首屏做营销式 landing page。

## 体验原则

- 首屏必须是可操作 App。
- 中文 UI 文案简洁明确。
- Apple 风格服务于层级、清晰度和触控，不做空洞装饰。
- iPhone/iPad 核心路径必须可触摸完成。
- Mac 专属增强不得成为移动端核心路径。
